Un juego 2D donde juegas como Jesús, coje cruces, evita morir por caída o por las bolas de fuego demoniacas. El juego incluye poderes como el de parar el tiempo con el RELOJ DE ARENA y un escudo activando el BLESSER.\hypertarget{index_autotoc_md1}{}\doxysubsection{\texorpdfstring{Controls}{Controls}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{Arrow Keys}}\+: Para mover al personaje de Derecha a Izquierda
\item {\bfseries{Down Key}}\+: Para bajar al personaje a la plataforma de abajo (o al vacio)
\item {\bfseries{Spacebar}}\+: Saltar
\item {\bfseries{B}}\+: Activate Blesser shield (if available)
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Game Features}{Game Features}}\label{index_autotoc_md2}

\item {\bfseries{\doxylink{classDemon}{Demon}}}\+: Crea bolas de fuego que hacen aparecer fuegos a todo lo que tocan.
